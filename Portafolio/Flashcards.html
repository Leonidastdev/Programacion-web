Pregunta

¿Qué es github?

Respuesta

Github es un portal creado para alojar el código de las aplicaciones de cualquier desarrollador.

Pregunta

¿Que es diseño responsive?

Respuesta

El diseño web responsive o adaptativo es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde ordenadores de escritorio a tablets y móviles.

Pregunta

¿Que son las etiquetas en HTML?

Respuesta

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo.
